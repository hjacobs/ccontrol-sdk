LDX - Load Index Register from Memory ;flags(N,Z)

LDX #opr ;X <= opr ;(IMM) 8 bit immediate

LDX opr ;X <= (opr) ;(DIR) 8 bit direct

LDX opr ;X <= (opr) ;EXT) 16 bit extended

LDX ,X ;X <= (X) ;(IX) indexed no offset

LDX opr,X ;X <= (opr + X) ;(IX1) indexed w/8 bit offset

LDX opr,X ;X <= (opr + X) ;(IX2) indexed w/16 bit offset
