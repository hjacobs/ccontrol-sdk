BRSET - Branch if Bit n in Memory Set ;flags(C)

BRSET n,opr,rel ;PC <= PC + 3 + rel ;(DIR) 8 bit Direct

;if Mn = 1 (n=0,1,..7)

Tests bit n(n=7,6,..0) in location M(opr) and branches if the bit is set. M can be any RAM or I/O register address in the $0000 to $00FF area of memory (direct addressing).
