BIT - Bit Test Acc with Memory ;flags(N,Z)

BIT #opr ;A ^ opr ;(IMM) 8 bit immediate

BIT opr ;A ^ (opr) ;(DIR) 8 bit direct

BIT opr ;A ^ (opr) ;(EXT) 16 bit extended

BIT ,X ;A ^ (X) ;(IX) indexed no offset

BIT opr,X ;A ^ (opr+X) ;(IX1) indexed w/8 bit offset

BIT opr,X ;A ^ (opr+X) ;(IX2) indexed w/16 bit offset

Neither the contents of A nor M are affected.
